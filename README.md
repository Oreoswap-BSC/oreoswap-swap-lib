# Oreoswap Swap Lib


Solidity libraries that are shared across Oreoswap contracts. These libraries are focused on safety and gas efficiency.

## Install

Run `yarn` to install dependencies.

## Test

Run `yarn test` to execute the test suite.

## Usage

Install this in another project via `yarn add @oreoswap/oreoswap-swap-lib` 

Then import the contracts via:

```solidity
import "@oreoswap/oreoswap-swap-lib/contracts/access/Ownable.sol"; 
```
